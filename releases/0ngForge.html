0ngForge v1.0.0

Released on 12 June 2026

Initial release of the 0ngForge offline VS Code installer packager.

  • Bundles VS Code binaries and selected extensions into a single self-contained archive for offline deployments.
  • Automates software setup and provisions VS Code extensions with a single execution.
  • Ensures extension dependency resolution and manifest validation.
